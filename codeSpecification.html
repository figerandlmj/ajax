HTML 1.语法 两个空格代替制表符（tab） 嵌套元素缩进一次（两个空格） 属性定义用双引号（""） 自闭合元素末尾不加斜线（/） 不要省略可选的结束标签 2.HTML5 doctype 每个html页面添加标准模式的声明（） 3.语言属性 为html根元素指定lang属性（en;en-us;zh;zh-CN;zh-cmn-Hans） 4.IE兼容模式 设置meta通知IE采用其所支持的最新的模式（） 5.字符编码 6.引入CSS和JavaScript文件 不需要指定type 7.使用为王 尽量使用最少的标签并保持最小的复杂度 8.属性顺序 class id name data-* src for type href value title alt role(role="button") arial-* 9.布尔型属性 布尔型属性在声明时不用赋值 disabled checked selected 10.js生成的标签 尽量避免js生成标签 CSS 1.语法 两个空格代表制表符（tab） 单独的选择器单独放在一行 在左花括号 { 前添加一个空格，右花括号 } 单独一行 : 后插入一个空格 所有声明语句以 ; 结尾 , 后插入一个空格（box-shadow:0 1px 2px #ccc, inset 0 1px 0 #fff;） rgb(),rgba(),hsl(),hsla(),rect()里面的 , 不加空格（background-color:rgba(0,0,0,.5)） 省略小数点前面的0（.5,-.5px） 16进制数全部小写，尽量使用简写形式的16进制（#fff） 为选择器中属性添加双引号（input[type="text"]） 避免为0指定单位（margin:0） 2.声明顺序 positioning 定位 position: absolute; top: 0; right: 0; bottom: 0; left: 0; z-index: 100; box model 盒模型 display: block; float: right; width: 100px; height: 100px; typographic 排字上的 font: normal 13px "Helvetina", sans-serif; line-height: 1.5; color: #333; text-align: center; visual 视觉的 background-color: #f5f5f5; border: 1px solid #e5e5e5; border-radius: 3px; misc 杂项 opacity: 1; 3.不要使用@import @import指令慢，并增加了额外请求 将 替换 4.媒体查询的位置 放在尽可能相关规则的附近 5.带前缀的属性 通过缩进的方式，让每个属性的值在垂直方向对齐 .selector{ -webkit-box-shadow: 0 1px 2px rgba(0,0,0,.15); box-shadow: 0 1px 2px rgba(0,0,0,.15); } 6.单行规则声明 如果只包含一条语句声明，则将语句放在同一行 7.简单形式的属性声明 合理运用简写形式，避免不必要的覆盖 8.Less和Sass中的嵌套 避免不需要的嵌套 9.Less和Sass中的操作符 在数学计算表达式的数值、变量和操作符之间均添加一个空格 10.注释 /* 良好的注释习惯 */ 11.class命名 class名称中只能出现小写字符和破折号（-） 避免过度的简写 尽可能短但意义明确 使用有意义的名称 基于最近的父class或基本class作为新class的前缀 使用.js-* 的class来标识行为，不要将这些包含到css文件 12.选择器 对于通用元素使用class 对于经常出现的组件，避免出现属性选择器 选择器尽可能短，建议不超过3个 只有在必要的时候才将class限制在父元素内 13.代码组织 以组件为单位组织代码段 制定一致的注释规范 使用一致的空白符将代码分割成块 如果使用了多个css文件，将其按照组件而非页面的形式分拆 14.编辑器配置 用两个空格代表制表符 保存文件时，删除末尾的空白符 设置文件编码为UTF-8 在文件末尾添加一个空白行
